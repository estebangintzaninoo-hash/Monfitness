⚡ Principe clé — HIIT post-muscu
Le HIIT après la muscu attaque la graisse avec les réserves de glycogène déjà épuisées → votre corps puise directement dans les graisses abdominales. C'est plus efficace que le cardio séparé pour votre objectif.

1 shake/jour · 30g · dans les 30 min post-séance
Priorité absolue après l'entraînement. Le HIIT + muscu vident les réserves — le shake relance immédiatement la synthèse musculaire. Ajouter une banane pour les glucides rapides post-effort.
⚡ Créatine monohydrate
5g/jour · tous les jours (repos inclus)
Phase de charge (optionnel) : 20g/jour × 5 jours, puis 5g/jour.
Timing : N'importe quand — mélangez dans l'eau, le shake ou le jus.
Pourquoi : Maintient la force et l'intensité pendant le HIIT — essentiel en déficit calorique.
